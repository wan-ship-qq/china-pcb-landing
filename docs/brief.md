# Бриф

## Проект
Лендинг для заказов PCB, PCBA-монтажа и компонентов из Китая в Россию.

## Цель
Получать заявки с Gerber-файлами и контактами клиента.

## Основной CTA
Загрузить Gerber файлы / Отправить заявку.

## Аудитория
Инженеры, радиолюбители, малый бизнес, разработчики электроники, которым нужны платы или компоненты из Китая.

## Tone of voice
Технологично, уверенно, коротко, без лишней «рекламности».
